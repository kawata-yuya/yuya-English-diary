# このレポジトリについて
川田侑弥の日常的な日記を英語で書き,英語に慣れるだけのレポジトリ.
恩師の英語の先生にだいぶ前から英語の3行日記を勧められており,
突然やる気になったかもしれないため,始めたいと思います.

# ルール
- 毎日せめて1行は英語で書くこと.
- 英語が嫌なら10文日本語で書くこと.
- 公開レポジトリとすること.
